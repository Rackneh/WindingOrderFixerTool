Get Started

  1. Import the package into your Unity project.
  2. A start screen window will show up. If it doesn't, open it at main menu Tools/RuntimeFbx/Start Screen
  3. Read this documentation and try the demos.
  4. Use RuntimeFbx APIs in your application.
  5. If you need help or want to make a suggestion, contact Support.

API

  • Create fbx file from a Mesh 
    MeshFilter meshFilter = gameObject.GetComponent<MeshFilter>();
Mesh mesh = meshFilter.sharedMesh;
RuntimeFbx.Util.SaveMesh(mesh, "D:\\mesh.fbx");

  • Load fbx file into a Mesh 
    Mesh mesh = RuntimeFbx.Util.LoadMesh("D:\\mesh.fbx");
MeshFilter meshFilter = gameObject.GetComponent<MeshFilter>();
meshFilter.sharedMesh = mesh;
